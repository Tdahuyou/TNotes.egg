## 1. 📒 概述

- 这篇笔记介绍了基于 Egg.js 搭建一个简单的 "Hello World" 示例的详细步骤，从零开始逐步完成。
- 我们将通过命令行工具创建项目，并实现一个基本的 API 接口。

## 2. 💻 准备必要的环境

在开始之前，请确保你的开发环境满足以下要求：

1. 已安装 Node.js（建议使用 LTS 版本，如 v16 或 v18）。
2. 已安装 `npm` 或 `yarn` 或 `pnpm` 包管理工具。（或者其他的包管理工具）

可以通过以下命令检查版本：

```bash
node -v
npm -v
```
