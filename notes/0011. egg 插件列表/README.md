## 2. 📒 Egg.js 插件命名规范

- 从插件列表中罗列的插件命名来看，会发现它们大多都是 `egg-xxx` 开头，然后后面跟一个插件名称，比如：
  - 静态资源映射的插件 `egg-static`
  - MySQL 插件 `egg-mysql`
  - ……
